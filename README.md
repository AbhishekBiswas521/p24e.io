Running locally
---------------

make sure you have Ruby installed, then run:

* `bundle install`
* `jekyll serve`

or, if you don't have Ruby installed:

`docker run --rm -v "$PWD":/srv/jekyll -p 4000:4000 jekyll/jekyll`

The site is running on http://localhost:4000/
